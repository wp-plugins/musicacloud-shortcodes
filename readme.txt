Embed your tracks from MusicaCloud to your wordpress project

== Description ==

This plugin let you to embed all Musicacloud Shortcodes into embeddable Musicacloud players. It works for any MusicaCloud track. Once you install this plugin, just copy and paste wordpress shortcodes from Musica Cloud to your editor.

== Installation ==

Upload the MusicaCloud ShortCode plugin to your blog, Activate it, then copy and paste your shortcode from musicacloud to your Articles or Pages into Wordpress.
